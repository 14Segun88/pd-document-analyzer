## 🔧 Технические детали

### Как работает Mistral 14B Reasoning

Модель возвращает данные в двух полях:
1. `reasoning_content` - рассуждения модели (CoT)
2. `content` - финальный ответ (JSON)

**Пример ответа:**
```json
{
  "choices": [{
    "message": {
      "reasoning_content": "I need to extract...",
      "content": "```json\n{\"title\": \"...\", ...}\n```"
    }
  }]
}
```

### Код извлечения (строки 261-263):
```python
msg = response.json()['choices'][0]['message']
result_text = msg.get('content') or msg.get('reasoning_content', '')
```

## ⚠️ Проблемы

### 1. Timeout на больших документах
**Причина**: Документы >5000 символов требуют больше времени
**Решение**: 
- Увеличить timeout до 180-240 сек
- Или обрезать текст до 3000 символов

### 2. Некоторые документы не обрабатываются
**Причина**: LLM timeout или ошибки парсинга JSON
**Решение**: Добавить retry с exponential backoff
