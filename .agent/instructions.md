# Инструкция для ИИ-агентов (AI Agent Instructions)

Внимание! Данный проект использует централизованную систему управления правилами и памятью.

**ПЕРЕД НАЧАЛОМ РАБОТЫ:**
1. Обязательно изучите содержимое директории `AGENTS_RULES/`.
2. Ознакомьтесь с `AGENTS_RULES/rules_general.md` для понимания стандартов проекта.
3. Прочтите `AGENTS_RULES/PROJECT_MEMORY.md`, чтобы увидеть историю успехов, ошибок и текущих оптимизаций.
4. Следуйте протоколу передачи смены в `AGENTS_RULES/session_handover.md`.

Это обеспечит преемственность контекста и предотвратит повторение ошибок.
