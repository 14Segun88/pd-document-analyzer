# Чеклист задач: Интеграция Ollama и нового Frontend (Vue/React)

## Общая Подготовка
- [ ] Согласовать с пользователем фреймворк для UI (Vue 3 / React)
- [ ] Проверить наличие `ollama` на ПК пользователя

## Этап 1: Backend (LLM Extraction)
- [ ] Установить локально модель `qwen2.5:3b` (или `llama3.2:3b`) через Ollama
- [ ] Подготовить `web_app_v5.py` (чистое Flask API)
- [ ] Написать функцию `extract_text_pdf()` и `extract_text_xml()`
- [ ] Сформировать жесткий системный промпт (Prompt Engineering) для LLM с форматом вывода JSON
- [ ] Написать логику `requests.post('http://localhost:11434/api/generate')`
- [ ] Убедиться, что Regex удален и поля (Название, Заказчик и т.д.) правильно возвращаются из LLM.

## Этап 2: Frontend (Vue/React)
- [ ] Инициализировать проект фронтенда (`npx create-vite@latest frontend --template vue`)
- [ ] Создать компонент `Uploader.vue` со строгим перехватом `.preventDefault()` для корректного Drag & Drop
- [ ] Связать Uploader с бэкендом через нативный `fetch()`
- [ ] Добавить состояние "Загрузка..."
- [ ] Создать компонент `Results.vue` для рендера массива пунктов "Содержимое" и табличных полей ответа JSON.

## Этап 3: Финальное тестирование
- [ ] Прогнать тестовый `ПД1-ПЗ.xml`
- [ ] Прогнать тестовый `ПД3-АР.pdf`
- [ ] Упаковать запуск в один скрипт (запуск Flask backend + Vite frontend сервера).
