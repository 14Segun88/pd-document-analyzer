## Ключевые паттерны

### Двухпроходный пайплайн
```python
# Pass 1: object_name
KB Override (score > 0.90) → Regex → LLM Pass 1 → Regex fallback → "Не определено"

LLM Pass 2 (с object_name) → KB Override (4 поля) → Результат
```

### Regex для object_name
```python
# Паттерны в тексте
r'(?:объект\s*(?:капитального\s*строительства)?\s*[:\-–]\s*)([А-ЯЁ][а-яёА-ЯЁ0-9\s«»\-,]+)'
r'(?:наименование|название)\s*(?:объекта)?\s*[:\-–]\s*([А-ЯЁ][а-яёА-ЯЁ0-9\s«»\-,]+)'

# Специфичные объекты
r'(МБОУ\s+СОШ\s+на\s+\d+\s+мест[а-яё]*)'
r'(многоквартирный\s+жилой\s+дом)'
r'(блочно-модульная\s+котельная)'

# Коды в filename
'МКД' → 'Многоквартирный жилой дом'
'БМК' → 'Блочно-модульная котельная'
'СОШ' → 'МБОУ СОШ'
```
