### Методы KB matching:
- `_calculate_score(filename, kb_title)` - расчёт с boost'ами
- `_get_kb_matching_entries(filename, top_n)` - топ-N похожих записей
- `_get_kb_direct_fields(filename)` - KB override если >60%

### Boost'ы для ключевых слов:
- АР/КР/ПБ/ТЗ/ИОС/ПОС/ОДИ → 75%
- ИОС5.3 + 5.5.3 → 95%
- ИОС5.3 + ИОС5 → 90%

### Визуализация на сайте:
- Badge: ✓ KB Override (≥90%) / ⚠ KB Match (60-89%) / ✗ Regex (<60%)
- Score в %
- Название записи из KB
- Топ-2 примера для LLM

### Известные проблемы:

- Если KB override <60% → используется regex (может быть неточно)
